alab_experiment_helper.tasks.dispensing module¶

dispensing(samples, input_file_path)[source]¶

Dispense samples according to the given recipes in .csv format.

The number of input samples should be equal to the number of recipes * replicates.

Parameters

  • samples (List[Sample]) – the samples to be operated on, in this setting, each sample is a crucible

  • input_file_path (Union[str, Path]) – the path to the input file, which is a csv file.
